矩阵链乘法总结：
1、什么是矩阵链？
	按我的理解，矩阵链就是将矩阵的维度存放在一个数组中，每相邻两个数字代表
	一个矩阵。因此相邻两个矩阵是可以相乘的。

2、矩阵链乘法需要解决什么问题？
	由于矩阵相乘的次序不同，往往计算量就不同。因此矩阵链乘法就是要寻找出计
	算量最小的计算方式，也就是使用括号改变结合的优先级。

3、怎样计算？
	根据问题的性质，我们可以将问题分成两个子问题求解，关键在于子问题如何划
	分。我们可以假设存在这么一个位置，将矩阵链分为左右两部分，该位置满足将
	左右两部分结合后问题得到最优解。子问题划分完成，那么现在还有一个问题：
	如果知道左右两部分的最优解，那么如何求得整体的最优解？这个问题简单，那
	就是将左边的解与右边的解相加，在加上左右结合所需的计算量就是整个问题的
	解。

4、如何构造最优解？
	我们将每次的划分点都记录在一个数组中s中，s[i][j]表示第i个矩阵到第j个矩
	阵能得到最优解的划分位置。那么就能得到这么一个事实：记k = s[i][j]，那么
	s[i][k]是左边最优解的划分位置，s[k+1][j]是右边最优解的划分位置；当i==j
	时，不能划分，于是输出该位置处的矩阵。因此我们可以递归构造最优解。
